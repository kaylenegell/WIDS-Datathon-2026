# Reference Documentation

This folder contains reference materials, notes, and documentation for the WiDS Datathon 2026 competition.

## Contents

### Competition Information
- **competition-overview.md** - Competition rules, timeline, submission requirements
- **evaluation-metrics.md** - Detailed explanation of C-index, Brier score, and hybrid metric
- **survival-analysis-notes.md** - Key concepts about survival analysis and censored data

### Feature Documentation (`feature-analysis/`)
Detailed analysis of all 34 features organized by category:

- **growth-features.md** - Fire expansion metrics (10 features)
- **kinematics-features.md** - Fire movement patterns (5 features)
- **distance-features.md** - Proximity to evacuation zones (9 features)
- **directionality-features.md** - Approach vectors (4 features)
- **temporal-metadata-features.md** - Timing context (3 features)

### Strategy & Insights
- **modeling-strategies.md** - Approaches, techniques, and lessons learned
- **data-insights.md** - Key patterns and correlations discovered

## Purpose

This documentation helps both team members:

- Share insights and strategies
- Maintain consistent understanding of the problem
